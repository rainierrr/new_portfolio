# Portfolio
## 利用技術
- React (Hooks)
- Typescript
- Materal-UI
- Docker

## 環境構築方法
1. Dockerとdocker-composeのインストール

2. イメージビルド
```
docker-compose build
```

3.　コンテナ作成
```
docker-compose up -d
```
4. コンテナの起動確認
```
docker-compose ps
```

5. ポートフォリオサイトにアクセス
localhost:3030にアクセスしてください。
ポートフォリオサイトを見ることができます。
